# HomeIQ Project Domains

This file defines the business domains for the HomeIQ IoT/home automation project and their primary experts.

## Project: HomeIQ - AI-Powered Home Automation Intelligence Platform

A comprehensive home automation system that transforms Home Assistant into an intelligent automation powerhouse with conversational AI, pattern detection, and advanced analytics.

### Domain 1: IoT & Home Automation

- **Primary Expert**: expert-iot
- **Description**: Home Assistant integration, device management, automation rules, smart home protocols, device communication
- **Key Areas**: Home Assistant WebSocket API, device protocols (Zigbee, Z-Wave, Matter), automation triggers, device health monitoring, device classification

### Domain 2: Time-Series Data & Analytics

- **Primary Expert**: expert-time-series
- **Description**: Time-series data storage, event processing, analytics, data retention, query optimization
- **Key Areas**: InfluxDB patterns, event ingestion, time-series queries, data aggregation, performance optimization

### Domain 3: AI & Machine Learning

- **Primary Expert**: expert-ai-ml
- **Description**: Conversational AI, pattern detection, automation mining, device intelligence, predictive analytics
- **Key Areas**: LLM integration, pattern mining, automation suggestions, device classification, anomaly detection

### Domain 4: Microservices Architecture

- **Primary Expert**: expert-microservices
- **Description**: Service orchestration, inter-service communication, distributed systems, service health monitoring
- **Key Areas**: Service boundaries, API design, service discovery, health checks, dependency management

### Domain 5: Security & Privacy

- **Primary Expert**: expert-security
- **Description**: Home automation security, device authentication, data privacy, secure communication, access control
- **Key Areas**: Device security, API authentication, data encryption, privacy compliance, secure protocols

### Domain 6: Energy Management

- **Primary Expert**: expert-energy
- **Description**: Energy consumption tracking, power optimization, energy pricing, carbon intensity, smart grid integration
- **Key Areas**: Energy monitoring, power consumption analysis, pricing optimization, carbon footprint tracking

### Domain 7: Frontend & User Experience

- **Primary Expert**: expert-frontend
- **Description**: React/TypeScript dashboards, real-time UI updates, interactive visualizations, user experience design
- **Key Areas**: React patterns, TypeScript best practices, real-time data visualization, responsive design, dashboard UX

### Domain 8: Home Assistant

- **Primary Expert**: expert-home-assistant
- **Description**: Home Assistant platform integration, REST API, WebSocket API, entity management, state management, automation YAML, integrations, custom components
- **Key Areas**: Home Assistant REST API, WebSocket event streaming, entity states, service calls, automation rules, integration patterns, custom component development
