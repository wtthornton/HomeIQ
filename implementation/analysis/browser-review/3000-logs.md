# Port 3000 – Logs Tab

**URL:** `http://localhost:3000/#logs`  
**Title:** HomeIQ Dashboard  
**Reviewed:** 2026-02-28

---

## Page summary

- **Purpose:** View recent logs (e.g. from admin-api container logs or aggregated service logs) for debugging.
- **Key UI:** Log viewer with service filter, level (info/warn/error), time, message; may support search and tail.

---

## Issues

| # | Severity | Description |
|---|----------|-------------|
| 1 | **High** | **Secrets in logs** – Ensure log content is sanitized (no API keys, tokens, passwords); redact or omit sensitive fields. |
| 2 | Medium | **Volume** – Default to last N lines or short time window; provide “Load more” or pagination to avoid huge payloads. |
| 3 | Medium | **Service filter** – If multiple services, filter must work and show which service(s) are selected. |
| 4 | Low | **Copy** – Provide “Copy” or “Copy selected” for log lines to paste into tickets. |

## Enhancements
